# 🍲 Jejak Rasa Nusantara

Aplikasi web berbasis PHP yang dirancang untuk menjelajahi, mendokumentasikan, dan menyimpan informasi kuliner tradisional dari berbagai penjuru Nusantara. Proyek ini dibuat untuk mempermudah pengguna dalam menemukan cita rasa khas Indonesia.

🌐 **Deployment:** [jejakrasa.site.je](https://jejakrasa.site.je)

---

## 🚀 Fitur Utama

* **Eksplorasi Kuliner:** Menampilkan daftar makanan khas daerah dengan informasi detail.
* **Sistem Autentikasi:** Fitur Login dan Logout yang aman untuk pengguna.
* **Dashboard Pengguna:** Halaman khusus bagi pengguna yang sudah masuk untuk mengelola aktivitas mereka.
* **Pencarian Berbasis Ajax:** Fitur pencarian makanan secara *real-time* tanpa perlu memuat ulang halaman (*reload*).
* **Filter Kuliner:** Mempermudah pencarian makanan berdasarkan kategori atau daerah tertentu.
* **Simpan Favorit:** Pengguna dapat menyimpan makanan kesukaan mereka ke dalam daftar favorit.

---

## 🛠️ Teknologi yang Digunakan

* **Backend:** PHP (Native)
* **Frontend:** HTML5, CSS3, JavaScript (Ajax)
* **Database:** MySQL
